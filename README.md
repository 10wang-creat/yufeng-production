# 昱鋒生產管理系統

離線版生產管理工具，涵蓋計件登記、庫存管理、加工進度追蹤與月報表產生。

## 功能

- 📊 **儀表板** — 總覽產量、金額、趨勢圖、員工排行
- 🔧 **計件登記** — 新增/查詢每日加工紀錄，自動計算金額
- 📦 **庫存管理** — 進貨、銷貨、庫存異動追蹤
- 🏭 **加工進度** — 毛胚→車床→鑽孔→銑床→熱處理→噴砂→染黑→磨刃口→成品
- 📋 **月報表** — 自動產生主管月報，支援列印
- 💾 **資料管理** — JSON 匯出/匯入備份

## 使用方式

直接開啟 `index.html` 即可，不需要網路連線、不需要安裝任何軟體。

## 技術說明

- 純前端 HTML/CSS/JavaScript，單一檔案
- 資料存於瀏覽器 localStorage
- 原始資料來自 Microsoft Access (.mdb) 匯出
- 支援 JSON 格式匯出/匯入備份
